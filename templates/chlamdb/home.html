Content of the database & main features

    {% if biodb == "chlamydia_04_16"%}
  • {{n_genomes}} Chlamydiae genomes from 26 different species (see the phylogeny)
    • {% elif biodb == "2019_06_PVC"%}
  • {{n_genomes}} PVC genomes (see the phylogeny)
    • {% else %}
  • {{n_genomes}} PVC genomes (see the phylogeny)
    • {% endif %}
  • Orthologous proteins identified with OrthoFinder
  • Detailed annotation of proteins based on
    • KEGG (Kegg orthologs)
    • COG (clusters of orthologous groups)
    • STRING (protein-protein interactions)
    • PRIAM (enzymes)
    • TCDB (curated transporters annotation)
    • SwissProt (manually annotated proteins)
    • InterproScan (domains and signatures)
    • DOOR (operons)
  • Precomputed homology searches with RefSeq and SwissProt databases for each protein
  • Precomputed phylogenetic reconstructions of orthologous groups including the closest RefSeq homologs
  • Pfam and COG phylogenetic profiles allowing to identify chlamydial domains/proteins rarely found in other prokaryotes (see for example PF01823)
  • Identification of conserved gene neighborhood among distantly related Chlamydiae and PVC (see for example the paper from Dandekar et al. for the rationale behind this)
  • Multiple tools allowing comparative analyses of Chlamydiae genomes
  • References to scientific publications collected from multiple sources

Latest News

December 02, 2019

Updated references to litterature (reported in the "Pubmed" tab available on each locus page) using data from STRING version 11.

November 26, 2019

New feature: Proteins associated to the same Pubmed entry (data from PaperBlast) can be retrieved from the "PubMed" tab on each locus page. See for example papers associated to the chaperone Scc2/CT576 and the 9 C. trachomatis proteins associated to the publication from Fields et al. (Analysis of putative Chlamydia trachomatis chaperones Scc2 and Scc3 and their use in the identification of type III secretion substrates, J Bacteriol. 2005).

October 22, 2019

Updated references to litterature (reported in the "Pubmed" tab available on each locus page) using the PaperBlast database from September 25 2019.

June 2019

Release 2.0 of the database with 277 PVC genomes.
